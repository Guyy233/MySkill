# publish-skill-to-myskill

这是一个用于在 skill 完成后，将其整理并上传到 `MySkill` GitHub 仓库的 skill。

## 用途

帮助把本地 Codex skill 复制到 `MySkill_repo`，补充适合 GitHub 浏览的 `README.md`，并完成提交与推送。

## 适用场景

- 一个新 skill 已经完成，准备上传到 `MySkill`
- 一个已有 skill 更新后，需要同步到 `MySkill`
- 需要为 skill 自动补充仓库展示用的 `README.md`
- 需要把 skill 集合统一提交到 GitHub
