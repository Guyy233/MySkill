# 论文分析

将一篇英文或中文学术论文加工成两份中文 PDF：包含逐图讲解的详细带读手册，以及保留原图、双语图注和图解的逐句中英对照三色标记译文。

## 适用场景

- 对论文进行带读、精读或逐段讲解
- 制作带前置知识、术语解释和自测题的阅读手册
- 按原句顺序逐句忠实翻译，不漏句、不合并为摘要
- 译文正文只放原句与译句，不插入非图批注、总结或解释
- 以绿、黄、红三色标记忠实译出的结论、方法和局限句
- 从原 PDF 高保真提取正文图片，放回对应段落附近并翻译图注
- 逐图说明横纵轴、单位、图例、子图、关键趋势、主要信息和论证作用
- 用原生 LaTeX 排版公式、变量和希腊字母，减少乱码与错位

## 运行要求

- 输入为论文 PDF 或完整论文文本
- 生成 PDF 时需要 XeLaTeX、CTeX 和 XeCJK
- 模板默认使用 macOS 的 `PingFang SC`；其他系统可改为 `Noto Sans CJK SC` 等字体

## 主要文件

- `SKILL.md`：完整工作流、逐句翻译边界、三色规则和校验要求
- `assets/preamble.tex`：字体、公式、三色高亮与版式定义
- `assets/guide_template.tex`：带读手册骨架
- `assets/translation_template.tex`：逐句中英对照三色标记译文骨架
- `references/figure-handling.md`：图片提取、放置、逐图分析和验收规范
- `references/translation-fidelity.md`：句级对应、忠实翻译、禁止增删和验收规范
