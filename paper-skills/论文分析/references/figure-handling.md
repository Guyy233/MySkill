# 论文图片提取、放置与分析规范

本规范适用于《带读手册》和《中英对照三色批注译文》。目标不是制作论文的缩略摘要，而是让读者在对应正文附近看到原图、读懂图注，并知道图片能支持什么结论。

## 1. 建立图片清单

通读 PDF 时，为每张正文编号图片记录：

- 图号与页码；
- 正文第一次引用它的位置；
- 原始英文图注；
- 图片类型（曲线图、柱状图、散点图、热图、结构图、流程图、示意图、照片等）；
- 子图 `(a)(b)(c)` 的数量与含义；
- 横轴、纵轴、单位、刻度和范围（若有）；
- 图例、颜色、线型、标记、误差棒或置信区间；
- 图片主要支持的论文主张；
- 提取后的安全文件名，例如 `fig-01.png`。

先完成清单，再写最终文档。不能凭正文中的 “Fig. 2” 引用猜测图片内容。

## 2. 高保真提取原图

优先从 PDF 中提取原始嵌入图片；如果一张图由多个 PDF 对象组成，或直接提取会丢失文字、图例、矢量线条，就把源页面以 200–300 dpi 渲染后，按图片边界做确定性裁剪。

必须遵守：

- 不使用生成式图像工具重绘、补字、去噪、改色或“增强”论文图片；
- 保留完整坐标轴、刻度、图例、子图标识和误差棒；
- 只裁掉与图片无关的页边空白，不裁掉图注所需的图内标识；
- 等比例缩放，不拉伸；
- 文件名只使用 ASCII 字母、数字和连字符；
- 提取后逐张打开检查，不能只看命令退出码。

如果图片分辨率确实不足，在图解中说明“原 PDF 分辨率限制导致部分数值不可辨认”，不要猜测。

## 3. 放回正文的规则

- 图片放在正文第一次实质讨论它的段落之后，而不是只根据 PDF 原页坐标机械放置。
- 保持图号顺序，不能把所有图片集中到章节末尾或文末。
- 图片宽度优先使用 `0.92\linewidth`；过高图片同时限制高度并保持比例。
- 使用 `[htbp]` 作为默认浮动位置；若图片漂移跨过章节，用 `\FloatBarrier` 收束。
- 原文的多子图默认作为一个完整组合图插入，避免拆开后丢失共同图例或相对布局。

推荐结构：

```latex
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.92\linewidth,height=0.68\textheight,keepaspectratio]{figures/fig-01.png}
  \caption{\textbf{Original caption:} <英文原始图注>\\
           \textbf{中文图注：}<忠实中文翻译>}
  \label{fig:fig-01}
\end{figure}
\begin{figureanalysis}[图 1 解读]
<按下方字段分析>
\end{figureanalysis}
```

## 4. 每张图的强制分析字段

按图片实际情况覆盖以下字段，不适用时明确写“不适用”，不要悄悄跳过：

1. **图形与子图结构**：是什么类型的图；`(a)(b)(c)` 各自展示什么。
2. **横轴**：变量、单位、范围、线性或对数刻度；无横轴时写“无坐标轴”。
3. **纵轴**：变量、单位、范围、数值越大/越小代表什么；无纵轴时写“无坐标轴”。
4. **图例与编码**：曲线、颜色、线型、点形、阴影、误差棒分别对应什么。
5. **关键观察**：峰值、谷值、拐点、收敛、交叉、差距、异常和可读的重要数值。
6. **主要信息**：用一至三句话回答“这张图主要讲了什么”。
7. **论证作用**：它验证了哪个方法设计、假设或论文结论；与前后文有什么关系。
8. **证据边界**：哪些是图中直接可见，哪些是作者解释，哪些只是合理推断；是否存在分辨率、样本量、误差范围或对比设置方面的限制。

结构图、流程图和照片没有传统坐标轴时，改为解释模块、箭头、信息流、空间布局、颜色编码和标注对象。

## 5. 两份交付物的侧重点

《带读手册》：先告诉读者阅读顺序，再解释图的工程或科学直觉、它为什么出现在这里、它与公式和实验设计如何对应。

《中英对照三色批注译文》：保留原图和双语图注，紧接一个明确标注为“图解”的分析框。不要把整理者的分析伪装成作者原文。分析框中只对关键方法、结论和局限使用三色标注。

## 6. 最终验收

- 图片清单数量与两份 PDF 中的正文图片数量一致；
- 图号、子图顺序、原始图注和中文翻译对应正确；
- 图片没有重复、缺失、拉伸或裁掉坐标与图例；
- 所有坐标文字在 100% 缩放下可读；
- 图片位于相关正文附近，没有漂移到错误章节；
- 每张图都有实质性分析，并回答“主要讲了什么”；
- 没有编造不可辨认的数值、单位、显著性或因果解释。
