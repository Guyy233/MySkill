# 逐句忠实翻译与验收规范

本规范只约束《逐句中英对照三色标记译文》。译文的职责是忠实呈现论文，不负责带读、总结或解释。除原图、双语图注和明确标识的逐图分析外，不加入任何整理者内容。

## 1. 先建立句子清单

按论文真实阅读顺序记录所有需要翻译的句子，并给内部工作清单编号 `S001`、`S002`……。双栏论文必须结合页面渲染确认顺序，不能直接相信文本提取结果。

清单覆盖标题、作者与单位信息、摘要、正文、脚注、致谢、表题与表内文字、图注、结论、局限和未来工作。公式、数字、单位、引用、章节号、图表号与参考文献元数据也要纳入对应检查。出版社下载提示和许可证页脚不属于论文正文，除非用户要求照版复制。

## 2. 按句配对，不按段概述

对清单中的每个句子依次输出：

1. 一个 `origtext`，只放该英文原句；
2. 紧邻的一个中文译句块；
3. 然后才进入下一个原句。

不得先堆放整个英文段落，再用一个中文段落概括。一个英文长句可以拆成多个中文分句以符合中文语法，但这些分句仍共同对应同一个原句，并必须保留它的全部命题。不得把两个或更多英文句子压缩、融合成一个中文摘要。

## 3. 忠实性硬规则

- 保留否定、条件、让步、因果、比较、范围、时态、指代和逻辑连接。
- 准确保留 `may`、`might`、`could`、`suggest`、`likely`、`approximately` 等模态、证据强度和不确定性，不把“可能”强化为“证明”。
- 保留所有数字、正负号、单位、百分比、统计量、置信区间、显著性、样本量和超参数。
- 保留公式、变量、上下标、希腊字母、图表号、公式号和引用关系；正文叙述照常翻译。
- 术语前后一致。必要时首次使用“中文（English）”，但不得额外插入术语讲解。
- 不擅自修正作者的可疑表述、数据或公式。确有明显排印问题时仍忠实翻译；相关讨论只放入带读手册。

## 4. 禁止事项

译文正文不得出现：

- summary、意译式概括或“本段主要讲了……”；
- 译者按、背景知识、概念解释、方法评价、证据评价或延伸推论；
- “换句话说”“简而言之”“值得注意的是”等原文不存在的引导句；
- 段前导读、段后小结、三色说明框或全文三色内容汇总；
- 为了着色而截取半句、删去限定语，或只翻译所谓关键句。

三色只改变忠实译句的背景颜色，不产生新内容。无论是否着色，每个原句都必须完整翻译。

## 5. 唯一例外：论文图片

在首次实质讨论图片的位置放置源图、英文原始图注、忠实中文图注和“图 X 图解”。分析框标题只显示“图 X 图解”，不得添加“整理者分析”“译文唯一的附加分析”等说明性括号。逐图分析按 `figure-handling.md` 执行。图解必须与原句—译句配对明显分隔，不能延伸为对普通正文的批注。

## 6. 句级验收

交付前逐项销号句子清单，并完成以下检查：

- 每个原句后都有且只有对应译句，没有孤立原文或孤立译文；
- 原句顺序未改变，无漏句、重复、跨句合并或摘要替代；
- 抽查全部否定词、条件词、比较词、模态词、数字、单位、引用和公式指代；
- 搜索新增解释常用词，确认其若出现在译文中，确实对应一个可见的原文句子；
- 删除除逐图分析之外的所有整理者说明；
- 三色标记没有改变句子文本，也没有被当作翻译覆盖证明。

只有句级清单全部销号，译文才算完成。段落数量相近、语义大致覆盖或阅读流畅都不能替代这一验收。
