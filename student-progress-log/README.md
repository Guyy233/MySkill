# student-progress-log

这是一个用于维护学生项目中文进展日志的 skill。

## 用途

帮助整理学生项目中的日常进展、历史阶段总结、周报素材和 logbook 风格记录，并在需要时配合 Git 做备份。

## 适用场景

- 整理当天进展
- 补写历史阶段总结
- 生成周报素材
- 生成 logbook 风格记录
- 更新 `progress.md`
- 在记录后顺手做 Git 提交与 GitHub 备份

## 当前支持模式

- 日常记录模式
- 周报模式
- logbook 模式
